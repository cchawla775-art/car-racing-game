# Car Racing Game 🚗

A browser-based 2D Car Racing Game built using HTML, CSS, and JavaScript.

## Features

* 🚗 Player Car Control
* 🚘 Enemy Cars
* 💥 Collision Detection
* 🏆 High Score System (Local Storage)
* 🎯 Level System
* ❤️ Lives System
* 🪙 Coin Collection
* ⚡ Nitro Boost (Shift Key)
* 🌙 Day/Night Mode
* 🌧️ Rain Effect
* 💣 Explosion Animation
* 🎮 Speed Selection (Easy, Medium, Hard, Extreme)
* 🔄 Restart Game Functionality

## Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)

## Controls

| Key   | Action      |
| ----- | ----------- |
| ←     | Move Left   |
| →     | Move Right  |
| Shift | Nitro Boost |

## How to Run

1. Download or clone the repository.
2. Open `index.html` in your browser.
3. Select game speed.
4. Click **Start Game**.
5. Enjoy racing!
